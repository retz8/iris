The Breakdown

  • What it does: Converts an arbitrary provider-model combo name into a URL-safe slug by lowercasing, collapsing non-alphanumeric runs to dashes, then stripping any leading or trailing dashes left over from the replacement.
  • Key responsibility: Generates stable, deterministic identifiers for provider-model combinations in the gateway's opencode plugin — used to key config entries and route lookups without collision risk from special characters in model names like slashes, dots, or version suffixes.
  • The clever part: Each trim helper short-circuits with the original reference when no trimming is needed (return i === value.length ? value : value.slice(0, i)) — a deliberate identity return to avoid a heap allocation on the common path where a slug is already clean.
