Before scrolling: what does this do?

The Breakdown

  • What it does: Encrypts named secrets to disk using XOR with a username-derived SHA-256 key mask, wraps each retrieved value in a SecretStr that redacts itself in repr based on secret length, and persists the store to ~/ga_keychain.enc.
  • Key responsibility: Keeps API keys off environment variables and out of log output for GenericAgent's tool calls — any LLM tool that needs a credential calls .use() to get the raw value without ever printing it.
  • The clever part: The four length tiers in SecretStr.__repr__ are calibrated to key type — short PINs show only ***, medium tokens show head/tail chars, long API keys show head+tail+length — enough to visually identify a key without exposing enough characters to reconstruct it.
